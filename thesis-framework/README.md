# Thesis Framework: Disruptive Innovation for Nigerian Markets

Distilled from *"Disruptive Innovation: The Path to Prosperity for Nations"* by Obinna Nwaokoro (NYU GLS, April 2025).

This is an operational knowledge base, not an academic summary. Use it to pressure-test product ideas, PRDs, and market entry strategies against proven disruptive innovation principles — with Nigeria-specific context.

## How to Use

- **Starting a new product idea?** Start with `framework/8-criteria-checklist.md`. Run your concept against all 8. If it fails multiple criteria, it may not be truly disruptive — which is fine, but know what you're building.
- **Writing or reviewing a PRD?** Use `validation/prd-review-checklist.md` for a structured pass.
- **Thinking about market entry?** Read `framework/market-entry-playbook.md` then `nigeria-context/` files for ground-level realities.
- **Need inspiration?** `case-studies/` shows the framework applied to real examples.
- **Want to kill a bad idea fast?** `validation/disqualification-criteria.md` tells you what disruptive innovation is NOT.
